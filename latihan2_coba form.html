Uji coba ini menunjukkan bahwa tombol reset akan mengembalikan semua input ke nilai defaultnya, sementara tombol kirim digunakan untuk mengirimkan data formulir. Atribut target dengan nilai _self berarti hasil formulir akan muncul di tab yang sama, sedangkan nilai _blank membuka hasil di tab baru. Perbedaan antara metode GET dan POST adalah GET mengirimkan data melalui URL, yang memungkinkan data terlihat, sedangkan POST mengirimkan data secara lebih aman tanpa menampilkannya di URL.
