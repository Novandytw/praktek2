Input dengan tipe checkbox memungkinkan pengguna memilih lebih dari satu opsi dalam satu grup. Ini berbeda dengan radio button yang hanya memungkinkan satu pilihan dalam satu grup.

Uji coba 4



Input dengan tipe color memungkinkan pengguna memilih warna, sedangkan date memungkinkan pemilihan tanggal. Selain itu, button dapat digunakan untuk menampilkan peringatan atau aksi lainnya.

Uji coba 5




Input dengan tipe datetime-local memungkinkan pengguna untuk memasukkan tanggal dan waktu lokal, month memungkinkan pemilihan bulan, dan email untuk memasukkan alamat email dengan validasi khusus.

Uji coba 6




Input bertipe file memungkinkan pengguna untuk mengunggah file dari perangkat mereka, memberikan cara mudah untuk berbagi data atau dokumen.

Uji coba 7





Atribut min dan max membatasi nilai yang bisa dimasukkan dalam input bertipe number dan range. Hal ini membantu pengguna untuk memilih atau mengetik nilai dalam rentang yang ditentukan.

Uji coba 8




Input url digunakan untuk memasukkan alamat web, search untuk pencarian, dan tel untuk memasukkan nomor telepon dengan validasi yang lebih spesifik.

Uji coba 9



Input time memungkinkan pengguna memilih waktu, sementara week memungkinkan pemilihan minggu dalam setahun. Kedua input ini membantu mempermudah proses pengisian data waktu dengan lebih tepat.
